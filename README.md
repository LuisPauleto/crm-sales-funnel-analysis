# crm-sales-funnel-analysis

CRM & Financial Analysis - CBS Dance Factory

Projeto de análise de dados desenvolvido com foco em análise financeira, CRM e funil de conversão utilizando Python, Pandas e visualização de dados.

Objetivo

Realizar tratamento, padronização e análise de dados financeiros e de CRM para geração de insights operacionais e acompanhamento de indicadores.

Tecnologias utilizadas
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab
Principais análises realizadas
Tratamento e limpeza de dados
Padronização de inconsistências em CRM
Análise temporal de faturamento
Análise de alunos por período
Construção de funil de conversão
Agrupamentos e métricas operacionais
Insights obtidos
Identificação de inconsistências cadastrais no CRM
Visualização do comportamento de faturamento mensal
Análise de evolução da base de alunos
Estruturação do funil de captação e conversão
Aprendizados

Durante o projeto foram aplicados conceitos de análise exploratória, tratamento de dados, regras de negócio e visualização de informações para suporte à tomada de decisão.
